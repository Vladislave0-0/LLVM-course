# LLVM-Сourse
Перед вами репозиторий с заданиями по дисциплине С. Лисицына "Современные методы разработки компиляторов" на 4 курсе ФРКТ МФТИ.

### Задания первого семестра:
1. **[Graphical application with SDL](/01_SDL/)**

    Реализация игры "Жизнь" Джона Конвея как простого графического приложения, основанного на `SDL 2.0`.

2. **[LLVM Pass](/02_LLVM_Pass/)**

    Это реализация LLVM пасса, который в runtime'е собирает трассу исполненных IR инструкций (`User <- Operand`) графического приложения.

3. **[LLVM IR Generator](/03_LLVM_IRGen/)**

    Это реализация IR-генератора графического приложения. После генерации приложения происходит его интерпретация с помощью `llvm::ExecutionEngine`.

    

4. **[Assembler to LLVM IR generator](/04_ASM2IR/)**

    Это реализация ассемблера, инструкции которого основаны на статистикие, собранной во втором задании. Ассемблер транслируется в `LLVM IR` двумя способами:
      - с вызовами эмулирующих функций (кроме инструкций потока управления).
      - с генерацией IR-эквивалентов для всех инструкций.

    После трансляции запускается игра "Жизнь".

5. **[Language to LLVM IR generator](/05_LANG2IR/)**

    Это реализация собственного C-подобного ,языка программирования. FrontEnd реализован на основе `ANTLR`. После трансляции из языка в `LLVM IR` вновь запускается игра "Жизнь".
