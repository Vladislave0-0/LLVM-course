# Language to LLVM IR generator

## Описание
1) Придумать грамматику своего ЯП, ориентируясь на графическое приложение и полученные в предыдущем задании `ISA` и `ASM`. 
2) Написать небольшие тесты для подмножеств вашей грамматики. Написать графическое приложение на этой грамматике.
3) Реализовать FrontEnd на `FlexBison`/`ANTLR` из ЯП в `LLVM IR`. Поддержать все написанные тесты и графическое приложение.
4) Подтянуть исполнение на `ExecutionEngine` полученного `LLVM IR` с графической библиотекой.
